# Noise

- Noise:一种简单且受限的语言来设计和构建安全通道协议,能够自动推断出复杂的密钥派生和本地状态转换,支持双向认证、前向保密、零往返加密、身份隐藏和其他高级功能.
- Noise Explorer:一个用于设计、推理、形式化验证和实现任意噪声握手模式的在线引擎

- 握手模式：由消息模式组合生成
